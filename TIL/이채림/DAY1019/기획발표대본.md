기존 서비스들이 많은데요. 

해당 서비스를 이용했던 사용자들은 다음과 같은 불편함을 겪었습니다.

기존 서비스들은 실제 경로를 반영하여 여행 동선을 제공하는 것이 아닌 직선형으로 동선을 제공합니다.

마지막으로 여행 경로 중간에 주변 여행지를 추천하는 것이 아닌 목적지 주변의 여행지를 추천합니다.

이러한 부분을 보완한 저희의 주요 기능은 다음과 같습니다.

# <장표 넘기기>

첫째, 그룹의 성향을 고려한 여행 경로/계획 추천 기능입니다.

둘째, 가고싶은 여행지 선택을 바탕으로 최적의 동선을 구성하는 기능입니다.

셋째, 이동 경로와 인접한 장소 중 가볼만한 장소를 추천하는 기능입니다.

기능 별로 상세히 소개드리겠습니다.

# <장표 넘기기>

성향 분석 기능입니다. 
여행 별로 그룹을 생성합니다. 그룹 생성 시 이번 여행에 추천 받고자 하는 취향을 입력 받습니다. 이 데이터는 추후 추천 여행지에 적용되어 여행 그룹 별로 원하는 성향에 맞는 여행지를 추천 받을 수 있습니다.  

# <장표 넘기기>

실제 경로를 바탕으로 한 최적의 동선을 제공합니다. 경로에 대한 시각화를 통해 사용자 중심의 경험이 가능해집니다. 

# <장표 넘기기>

경로와 인접한 주변 여행지를 추천 기능입니다.

사용자가 선택한 경로 리스트 사이에 플러스 버튼을 이용하여 주변 여행지 추천 페이지로 이동합니다. 해당 페이지에서 그룹 생성 시 선택한 취향에 맞는 여행지를 추천 받아 경로에 적용할 수 있습니다.
